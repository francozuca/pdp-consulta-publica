# Consulta Pública - Pueblos del Plata

Single Page Application (SPA) creada para informar a los vecinos de Pueblos del Plata sobre la consulta pública del Municipio de Berazategui y cómo participar votando a favor del proyecto.

## Archivos

- `index.html` - Página principal con toda la información
- `styles.css` - Estilos CSS vanilla
- `script.js` - JavaScript vanilla para funcionalidades interactivas

## Cómo usar

1. Abre `index.html` en tu navegador web
2. La página mostrará:
   - Información sobre la consulta pública
   - Por qué es importante participar
   - Instrucciones paso a paso para votar
   - Enlace directo al formulario oficial

## Características

- ✅ Diseño responsive (se adapta a móviles, tablets y escritorio)
- ✅ Contador de días restantes hasta el cierre de la consulta
- ✅ Animaciones suaves al hacer scroll
- ✅ Instrucciones claras y visuales
- ✅ Enlaces directos al formulario oficial

## Despliegue

Puedes subir estos archivos a cualquier servidor web estático o usar servicios como:
- GitHub Pages
- Netlify
- Vercel
- Cualquier hosting estático

Simplemente sube los tres archivos (`index.html`, `styles.css`, `script.js`) a la misma carpeta en tu servidor.
